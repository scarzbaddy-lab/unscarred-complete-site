====================================================
OVERVIEW
====================================================

When someone purchases a product, you want:
1. Instant notification to YOU (so you know to prep for coaching calls)
2. Welcome email to CUSTOMER (personalized by product type)
3. Optional: Add to email list, CRM, calendar, etc.

Payhip supports webhooks that fire immediately after purchase.


====================================================
OPTION 1: ZAPIER (EASIEST - RECOMMENDED)
====================================================

COST: Free tier (100 tasks/month) or $20/month for more

SETUP STEPS:

1. Create Zapier Account
   - Go to https://zapier.com
   - Sign up free

2. Create New Zap: "Payhip Purchase ‚Üí Email"
   
   TRIGGER:
   - Search for "Payhip"
   - Select "New Sale" as trigger
   - Connect your Payhip account (you'll get an API key)
   
   ACTION 1 - ALERT EMAIL TO YOU:
   - Add "Gmail" or "Email by Zapier"
   - Send email to: your@email.com
   - Subject: "New Purchase: {{Product Name}}"
   - Body: 
     """
     New sale on Unscarred!
     
     Product: {{Product Name}}
     Customer: {{Buyer Email}}
     Amount: {{Sale Total}}
     Time: {{Sale Date}}
     
     If this is a coaching session, reach out within 24 hours.
     """

   ACTION 2 - WELCOME EMAIL TO CUSTOMER:
   - Add another Gmail/Email action
   - Send to: {{Buyer Email}}
   - (See Welcome Email Templates below)

3. Turn On Your Zap


====================================================
OPTION 2: MAKE.COM (MORE POWERFUL)
====================================================

COST: Free tier (1000 ops/month) or $9/month

Better for:
- Complex routing (different emails per product)
- Connecting to more tools
- Higher volume

SETUP:

1. Create Make.com Account
   - Go to https://make.com

2. Create Scenario:
   
   MODULE 1: Payhip Webhook
   - Add "Webhooks" ‚Üí "Custom Webhook"
   - Copy the webhook URL Make gives you
   - Go to Payhip ‚Üí Settings ‚Üí Webhooks
   - Paste the URL
   - Select "sale.completed" event
   
   MODULE 2: Router (for different products)
   - Add Router module
   - Create routes based on product name/ID
   
   MODULE 3: Email for Each Route
   - Route 1: Coaching purchases ‚Üí coaching welcome email
   - Route 2: Workbooks ‚Üí workbook welcome email
   - Route 3: Free downloads ‚Üí lead nurture email


====================================================
OPTION 3: PAYHIP BUILT-IN + EMAIL SERVICE
====================================================

Use Payhip's native features + Mailchimp/ConvertKit/Mailerlite

SETUP:

1. In Payhip ‚Üí Settings ‚Üí Integrations
   - Connect Mailchimp, ConvertKit, or other
   - Enable "Add buyer to list after purchase"

2. In Your Email Service:
   - Create automation: "When tag added ‚Üí Send welcome email"
   - Create different tags per product category
   - Build welcome sequences

Payhip Tags You Can Use:
- Product name
- Product category
- Bundle vs single
- Price tier


====================================================
PAYHIP WEBHOOK CONFIGURATION
====================================================

1. Go to: https://payhip.com ‚Üí Settings ‚Üí Webhooks

2. Add Webhook URL:
   - For Zapier: Use the webhook URL from your Zap
   - For Make: Use the webhook URL from your scenario
   - For custom: Your server endpoint

3. Select Events:
   [x] sale.completed      (purchase made)
   [x] sale.refunded       (for your records)
   [ ] subscription.active (if you add subscriptions)

4. Webhook Payload (what you'll receive):
   {
     "event": "sale.completed",
     "data": {
       "buyer_email": "customer@email.com",
       "buyer_name": "Jane Doe",
       "product_name": "Anxious Healing Program",
       "product_id": "abc123",
       "sale_price": "147.00",
       "currency": "USD",
       "sale_date": "2026-01-04T14:30:00Z",
       "download_url": "https://payhip.com/..."
     }
   }

====================================================
PRODUCT-SPECIFIC ROUTING LOGIC
====================================================

Set up IF/THEN rules in Zapier or Make:

IF product name CONTAINS "Coach" OR "Session"
   ‚Üí Send Coaching Welcome Email
   ‚Üí Notify Scar immediately (text or priority email)
   ‚Üí Add "Coaching Client" tag

IF product name CONTAINS "Program" OR "Healing"
   ‚Üí Send Program Welcome Email
   ‚Üí Add "Program Student" tag
   ‚Üí Start 7-day check-in sequence

IF product name CONTAINS "Workbook" OR "Scripts" OR "Drills"
   ‚Üí Send Workbook Welcome Email
   ‚Üí Add "Workbook Buyer" tag

IF product price = 0 OR product name CONTAINS "Free" OR "Toolkit"
   ‚Üí Send Free Download Email
   ‚Üí Add "Lead" tag
   ‚Üí Start nurture sequence

IF product name CONTAINS "Text" OR "Crisis Week" OR "Best Friend"
   ‚Üí Send Text Support Email
   ‚Üí Add "Active Text Client" tag
   ‚Üí Add to active client tracking sheet
