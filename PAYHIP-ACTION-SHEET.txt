PART 2: CONVERT THUMBNAILS (10 minutes)
====================================================

Your thumbnails are SVG format. Payhip needs PNG.

[ ] Option A: Run PowerShell script
    - Open PowerShell in project folder
    - Run: .\convert-thumbnails.ps1

[ ] Option B: Use online converter
    - Go to: https://cloudconvert.com/svg-to-png
    - Batch upload all SVG files from thumbnails folder
    - Download PNGs

[ ] Option C: Convert in Canva (one by one if needed)
