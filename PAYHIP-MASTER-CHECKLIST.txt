====================================================
TROUBLESHOOTING
====================================================

CSS NOT SHOWING?
- Clear browser cache (Ctrl+Shift+R)
- Make sure CSS is pasted in Custom CSS, not Custom Code
- Check for error messages in browser console (F12)

PRODUCTS NOT APPEARING?
- Make sure products are Published (not Draft)
- Check category assignment
- Refresh store page

THUMBNAILS LOOK WEIRD?
- Use 1600x900 PNG images
- Make sure they're uploaded, not just linked

EMAILS NOT SENDING?
- Check spam folder
- Verify email settings in Payhip
- Contact Payhip support if persistent

CHECKOUT LOOKS WRONG?
- The checkout CSS should be in the same Custom CSS field
- Scroll to bottom of PAYHIP-CUSTOM-CSS.txt for checkout styles
