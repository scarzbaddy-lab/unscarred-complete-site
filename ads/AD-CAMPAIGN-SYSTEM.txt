UNSCARRED AD CAMPAIGN SYSTEM

Last Updated: January 4, 2026

====================================================
NAMING CONVENTION
====================================================

Format: GOAL-ANGLE-SIZE.html

GOALS:
  LEAD    = Get email/quiz signup
  SALE    = Direct offer/program sale
  TRUST   = Build credibility (testimonials, proof)
  BRAND   = Awareness, values, manifesto

ANGLES:
  Hook        = Pattern hook, emotional trigger
  Quiz        = Specific quiz promo
  Offer       = Program/coaching CTA
  Testimonial = Social proof
  Manifesto   = Brand philosophy
  Truth       = Reframe/education
  Symptom     = Pain point callout

SIZES:
  Vertical  = 1080x1920 (Stories, TikTok, Reels)
  Square    = 1080x1080 (Feed posts)
  Banner    = 1200x628 (Facebook, Twitter, LinkedIn)
  Leader    = 728x90 (Leaderboard/header)


====================================================
TAG SYSTEM
====================================================

TAG 1: AUDIENCE
------------------------------------------
  AX    = Anxious attachment
  AV    = Avoidant attachment  
  MX    = Mixed signals / confused
  ND    = Neurodivergent (ADHD, autism)
  TB    = Trauma bond / abuse survivors
  WAR   = Survival mode / complex trauma
  MEN   = Men specifically
  WMN   = Women specifically
  ALL   = Universal

TAG 2: PURPOSE
------------------------------------------
  HOOK  = Stop scroll, create curiosity
  QUIZ  = Drive quiz completion
  PROOF = Build trust with testimonials
  OFFER = Sell program/coaching
  BRAND = Establish identity/values


====================================================
DECISION RULES
====================================================

IF views drop → Post HOOK
   (Scroll-stopping content, emotional triggers)

IF clicks drop → Post QUIZ  
   (Clear value, specific outcome promised)

IF trust feels low → Post TESTIMONIAL
   (Real results, relatable quotes)

IF you need money now → Post OFFER
   (Coaching CTA, program promo)

IF engagement is flat → Post BRAND
   (Manifesto, philosophy, values)


====================================================
ROTATION FORMULA
====================================================

Weekly Content Mix (7 posts):
  2x HOOK (engagement)
  2x QUIZ (leads)
  1x TESTIMONIAL (trust)
  1x OFFER (sales)
  1x BRAND (identity)

Adjust based on metrics and goals.

====================================================
VARIANT SYSTEM (Soft / Sharp / Savage)
====================================================

For every winning ad, create 3 copy versions:

SOFT = Gentle, compassionate, understanding
  "You're not broken. You're running survival patterns."

SHARP = Direct, clear, no-BS
  "It's not a character flaw. It's a nervous system in survival mode."

SAVAGE = Confrontational, wake-up call
  "Stop calling yourself 'too sensitive.' Your hypervigilance kept you alive."

File naming for variants:
  LEAD-Hook-Patterns-Vertical-SOFT.html
  LEAD-Hook-Patterns-Vertical-SHARP.html
  LEAD-Hook-Patterns-Vertical-SAVAGE.html
