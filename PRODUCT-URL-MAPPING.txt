====================================================
HOW TO CREATE PRODUCTS IN PAYHIP
====================================================

1. Go to https://payhip.com/auth/login
2. Click "Add Product"
3. Choose "Digital Download" for workbooks
4. Choose "Course" for programs (if using course feature)
5. Set the URL slug to match the /l/xxx above
6. Upload the PDF/DOCX file
7. Set the price
8. Copy the final product URL
9. Mark as [x] in this document
10. Update website links if the URL differs
